# Stocks ERP

**Stocks ERP** is a simple and efficient web-based inventory management system for handling multiple stock types, transactions, and reporting. It supports both **Stock 1** and **Stock 2**, with features for receiving, dispatching, and monitoring product quantities in real-time.

---

## Features

- **Product Management**
  - Add, edit, and view products.
  - Auto-fill suggestions for product code, name, and loose code.

- **Stock Transactions**
  - Record **receiving** and **dispatch** entries for Stock 1 & Stock 2.
  - Supports batch entries with multiple items per transaction.
  - Auto-calculation of total quantities.

- **Reporting**
  - View current stock per product.
  - Generate transactions reports.
  - Export final stock and transactions as **CSV** or **Excel**.

- **Search & Filter**
  - Search by code, name, type, invoice, or party.
  - Filter transactions dynamically.

- **Frontend**
  - Responsive UI with separate pages for stock transactions and reports.
  - Real-time calculations for totals in transaction forms.

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL / MariaDB  
- **Utilities:** XLSX for Excel export
